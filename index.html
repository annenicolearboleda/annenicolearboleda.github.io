The Pig Game

Technologies used: JavaScript, HTML, CSS

Our game app is a thrilling dice rolling game where players aim to reach a final score of 100. Each dice roll adds to the player's temporary score, which can be added to their final score by using the "Hold" button or continued by rolling the dice.
