The Pig Game

Technologies used: JavaScript, HTML, CSS

This game app is a thrilling dice rolling game where players aim to reach a final score of 100. Each dice roll adds to the player's temporary score, which can be added to their final score by using the "Hold" button or be continued to increase by rolling the dice some more. But beware, rolling a 1 resets the temporary score to 0.
